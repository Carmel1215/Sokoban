# Sokoban

Sokoban은 2025학년도 북일고 학예술제를 위해 출품한 작품으로 시간제한이 존재하는 간단한 소코반 게임입니다.

## Installation

1. Releases에 들어갑니다.
2. 운영체제에 맞춰 zip 파일을 다운받습니다. (Windows: Sokoban.zip / Mac: Sokoban.app.zip)
3. 압축을 풀고 실행 파일을 실행합니다.

## Usage

인게임 내에 튜토리얼이 있습니다.
